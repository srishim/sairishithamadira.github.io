Real-Time Sign Language Detection

A web application that translates sign language gestures into text and spoken words using deep learning and MediaPipe for real-time hand tracking.

Key Features:
  • Real-time hand tracking with MediaPipe
  • CNN and FNN-based gesture recognition
  • High test and real-time accuracy (up to 98.96%)
  • Supports letters (A-Z) and numbers (0-9)
Technologies Used:
Python OpenCV MediaPipe TensorFlow Keras FNN
